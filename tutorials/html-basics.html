HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage.

Basic Structure

Every HTML document follows a basic structure:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <h1>Main Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Common Elements

HTML elements are the building blocks of web pages:

  • <h1> to <h6> - Headings (hierarchical structure)
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Divisions/containers
  • <span> - Inline containers

<!-- Headings -->
<h1>Main Title</h1>
<h2>Subtitle</h2>

<!-- Links and Images -->
<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Description">

<!-- Text Formatting -->
<p>This is <strong>bold</strong> and this is <em>italic</em>.</p>

Forms & Input Elements

Forms allow users to input data and submit it:


<form action="/submit" method="POST">
    <!-- Text Input -->
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>

    <!-- Password Input -->
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">

    <!-- Radio Buttons -->
    <input type="radio" id="option1" name="choice" value="1">
    <label for="option1">Option 1</label>

    <!-- Checkboxes -->
    <input type="checkbox" id="check1" name="subscribe">
    <label for="check1">Subscribe to newsletter</label>

    <!-- Select Dropdown -->
    <select name="country">
        <option value="us">United States</option>
        <option value="uk">United Kingdom</option>
    </select>

    <!-- Submit Button -->
    <button type="submit">Submit</button>
</form>

Lists & Tables

HTML provides different types of lists and table structures:


<!-- Unordered List -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>First Step</li>
    <li>Second Step</li>
</ol>

<!-- Table Structure -->
<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>

Semantic HTML

Semantic HTML elements provide meaning to the structure:


<header>
    <nav>
        <!-- Navigation content -->
    </nav>
</header>

<main>
    <article>
        <section>
            <h2>Article Section</h2>
            <p>Content here...</p>
        </section>
    </article>

    <aside>
        <!-- Sidebar content -->
    </aside>
</main>

<footer>
    <!-- Footer content -->
</footer>

Best Practices

  • Always use proper indentation for better readability
  • Include alt attributes for images
  • Use semantic elements when possible
  • Ensure proper nesting of elements
  • Keep your code clean and well-commented
  • Use meaningful class and ID names
  • Validate your HTML using W3C validator
